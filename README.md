  <h3>프로젝트 개요</h3>
  <hr>
  <p>
    HandBook은 차세대 소셜 네트워킹 플랫폼으로, 사용자 중심의 커뮤니케이션과 개인화를 제공합니다.<br>
    HandBook은 모든 사용자에게 안전하고 개인화된 소셜 네트워킹 경험을 제공하는 것을 목표로 하고 있습니다.
  </p>
